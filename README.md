# GETAI Core Bot

Autonomous AI assistant for GETAI IT Company.

## Features
- Local AI (Ollama qwen2.5:3b) - 0 tokens
- Groq fallback (llama-3.3-70b)
- PostgreSQL memory + summaries
- Multi-agent system (Orchestrator, Manager, Architect, Backend, Frontend, DevOps, QA)
- Jira integration
- Heartbeat autonomous task execution
- Server management via Telegram

## Stack
- Python + aiogram 3
- OpenAI-compatible Ollama endpoint
- PostgreSQL + pgvector
- Docker
